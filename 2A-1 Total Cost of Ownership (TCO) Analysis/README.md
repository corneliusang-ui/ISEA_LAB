# 2A-1 Total Cost of Ownership (TCO) Analysis

## Overview

To understand and apply the TCO methodology by comparing two printers over a specified period, incorporating fixed and variable costs, and interpreting the results for decision-making. 

---

## 5. Spreadsheet Calculation (TCO Model)

A Microsoft Excel spreadsheet was created to calculate the Total Cost of Ownership for both printers.

The spreadsheet contains:

- Quantity
- Unit Cost
- Total Cost
- Total Cost of Ownership (5 Years)

Formula used:

```
Quantity × Unit Cost = Total Cost
```
